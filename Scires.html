AUP

Bar Graph

a graph using bars to compare data from two or more categories

Biosafety

the application of knowledge, techniques and equipment to prevent exposure to biohazards

Euthanasia

act of killing someone painlessly

Frequency Distribution

an overview of all values in a variable and the number of times the said variable occurs

Histogram

a graph that uses large rectangular bars that shows continuous data

IACUC

Institutional Animal Care and Use Comitees // IACUC reviews research protocols and conducts evaluations of the institution's animal care and use

Line Graph

a graph using a line with dots presenting the change of number or value over time

Mean

represents the average of the value of the dataset

Median

the middle value of the dataset in which the dataset is arranged ascending or descending

Mode

the frequency of the occuring value in the dataset

OLAW

Office of Laboratory Animal Welfare // helps in developing the role of the IACUC

Pie Chart

a circular graph that shows proportional data

Qualitative Observation

data that doesn't use values; descriptive

Quantitative Observation

data that is based on values; percentage, values, etc.
